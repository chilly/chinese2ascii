Yes, this applet is inspired by Figlet fonts.  However it doesn't use any other font besides these GB-2312 compliant simplified chinese characters made with only the '#' (pound) character.  ASCII characters in the input will be removed before conversion (only Unicode chinese will be converted).

If the applet doesn't work in your browser, you may need to download the application version.  This is an executable jar file:  double-click or run this file on your computer after it downloads.  You will need to have a version of the JRE installed. 

Many thanks to the various authors of codeset such as EUC_TW, EUC_CN, GB2312-80, CNS, HZ, the makers of java encodings, and to Ngai Kim Hoong and Erik.

If you would like to see other encodings supported, are having problems with the converter, or can tell me something about related figlets for chinese, please email: info@javaspider.com. 

Good luck -
